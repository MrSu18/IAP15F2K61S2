# IAP15F2K61S2
蓝桥杯单片机组MCU库

>   -   第十四届蓝桥杯单片机组广东赛区一等奖
>   -   第十四届蓝桥杯单片机组全国总决赛二等奖

## 文件结构

-   蓝桥杯官方单片机资料：省赛国赛赛场会提供的资料包
-   software：板子考点模块的库工程
    -   libraries
        -   sushizhou_common：软件延时、数据类型定义
        -   sushizhou_device：外部模块芯片驱动
        -   sushizhou_drive：定时器、串口、中断、iic等
    -   user：存放工程的main.c和省赛练习题的main.c文件

-   国赛：国赛代码和国赛题目
-   省赛：省赛代码
